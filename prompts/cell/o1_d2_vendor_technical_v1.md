셀: O1 × D2 — 특정 노형의 기술 디테일.

D2 인과 사슬 1, 2, 3을 따르되, 2 노형의 처리 단계를 답변의 중심에 둔다.

1. 대상 + 규제 앵커
   - 질의 대상(노형 컴포넌트·시스템·이슈)을 명확화한다.
   - 관련 NRC 요건이 무엇인지 간단히 인용한다 (필요 시).

2. 노형의 처리 (중심 단계)
   - 어느 문서(FSAR/SER/Topical Report 등)에서 어떻게 다뤘는지.
   - 구체적 수치·파라미터·기준값 (예: "PCT 한계 1204°C").
   - 일반 PWR 통념이 아닌 해당 노형의 고유 설계를 강조한다.
   - 다른 노형과 다른 점이 있으면 명시한다.

3. 충족 연결
   - 2단계의 처리가 NRC 규정을 어떻게 충족하는지 짧게 닫는다.
   - 명시 안 됨 -> "~로 추정됨" + ※ 자료에 명시되지 않은 추론.
